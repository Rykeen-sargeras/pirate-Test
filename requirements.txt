# How Much of a Pirate Are You?

A Python (Flask) app. Spin 1-100%, get your pirate loyalty tier, and above
95% (SAVAGE) you get a funny story of how Savage rescued you.

Features:
- Name-hash results: same name always gets the same fate (SHA-256).
  Blank name = pure random.
- Sea shanty background music generated live with the Web Audio API
  (no audio files, fully copyright-free) with volume slider and mute.
- Scimitar mouse cursor.
- Modern dark glass UI with animated progress dial and drifting waves.

## Run locally

    pip install -r requirements.txt
    python app.py

Open http://localhost:5000

## Deploy on Railway

Only these files are needed (no Procfile, no railway.json):

    app.py
    requirements.txt
    templates/index.html
    static/style.css
    static/script.js

Railway auto-detects Python, runs pip install, and starts with
"python app.py" - app.py already binds 0.0.0.0 and reads $PORT.

After a green deploy: Settings > Networking > Generate Domain to get
your public URL.

IMPORTANT when editing on GitHub: line 1 of requirements.txt must be
"Flask==3.0.3". Nothing else goes in that file.
